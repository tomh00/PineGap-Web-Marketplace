Reflections on what you have learnt through the project and what you would do differently.

We learnt a lot through this project such as how to integrate a MYSQL database into a web application. We also learnt of the challenges with storing datasets because the datasets content can be quite varied e.g. it might include text, images etc. We learnt how to use external libraries effectively e.g. we used an external library to help with the login register functionality.

There are a few things we could do differently in the future including building mini deadlines into the project, committing more often using Trello to keep track of work to do, add support for images in the datasets. Another thing we could do is server side validation as we only used client side application in this project due to time constraints.

Another thing we could do is to make greater use of javascript in the project and overall make the website more interactive for the user.
